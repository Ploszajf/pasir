Aplikacja webowa Full-Stack do zarządzania budżetem osobistym oraz grupowym rozliczaniem wydatków.

Projekt łączy backend oraz frontend w jedno monorepo. Umożliwia rejestrację użytkowników, dodawanie transakcji, tworzenie grup oraz wspólne zarządzanie i zatwierdzanie długów wewnątrz grup.

### Architektura systemu:
* **Backend:** Aplikacja napisana w języku Java przy użyciu frameworka Spring Boot.
* **Frontend:** Aplikacja w React z wykorzystaniem Vite oraz TypeScript. Serwowany produkcyjnie za pomocą Nginx.
* **Baza danych:** MySQL 8.4.
* **Konteneryzacja:** Docker.
